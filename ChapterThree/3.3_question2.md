We are given:

$
|A| = 7
$
$
|B| = 11
$

We want to find:

$
|A \cup B| + |A \cap B|
$

Using the principle of inclusion and exclusion:

$
|A \cup B| = |A| + |B| - |A \cap B|
$

Substitute that into the expression:

$
|A \cup B| + |A \cap B| = (|A| + |B| - |A \cap B|) + |A \cap B|
$

Simplify the expression:

$
|A| + |B| - |A \cap B| + |A \cap B|
$

$
-|A \cap B| + |A \cap B| = 0
$

$
|A| + |B|
$

Now substitute the values:

$
= 7 + 11 = \boxed{18}
$
